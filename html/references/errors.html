nmrespy._errors¶

nmrespy-specific errors

exception nmrespy._errors.AttributeIsNoneError(attribute, method)[source]¶

Raise when the user calls a get_<attr> method, but the attribute is None

exception nmrespy._errors.InvalidDirectoryError(dir)[source]¶

Raise when the a dictionary does not have the requisite files

exception nmrespy._errors.InvalidUnitError(*args)[source]¶

Raise when the specified unit is invalid

exception nmrespy._errors.LaTeXFailedError(texpath)[source]¶

Raise when the user calls write_result(), with format set to ‘pdf’, but compiling the TeX file failed when pdflatex was called.

exception nmrespy._errors.MoreThanTwoDimError[source]¶

Raise when user tries importing data that is >2D

exception nmrespy._errors.NoParameterEstimateError[source]¶

Raise when instance does not possess a valid parameter array

exception nmrespy._errors.ParameterNotFoundError(param_name, path)[source]¶

Raise when a desired parameter is not present in an acqus/procs file

exception nmrespy._errors.PhaseVarianceAmbiguityError(mode)[source]¶

Raise when phase_variance is True, but ‘p’ is not specified in mode

exception nmrespy._errors.TwoDimUnsupportedError[source]¶

Raise when user tries running a method that doesn’t support 2D data yet
